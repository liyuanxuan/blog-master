可接受的日期格式

  • yyyy-MM-dd
  • yyyy/MM/dd
  • yyyy-MM-dd HH:mm:ss
  • yyyy-MM-dd HH:mm
  • yyyy-MM-dd HH
  • yyyy/MM/dd HH:mm:ss
  • yyyy/MM/dd HH:mm
  • yyyy/MM/dd HH

PageResult<T>默认属性

属性名 类型 是否可能为null 说明
datas List<T> 否 有查询结果而定，例如PageResult<Tag>，那么类型就是List<Tag>,如果没有结果，则为空集合
totalPage int 否 总页码
currentPage int 否 当前页码
pageSize int 否 每页记录数
totalRow int 否 总纪录数
listbegin int 否 分页起始页码
listend int 否 分页结束页码
param PageQueryParam 否 查询参数

一个简单的分页示例：

<div th:if="${page.totalPage > 1}">
	<ul class="pagination">
		<li
			th:each=" i : ${#numbers.sequence(page.listbegin, page.listend-1)}"
			th:class="${(i == page.currentPage) ? 'active' : ''}"><a
			href="href" th:text="${i}"></a>
	</ul>
</div>
